# 건강 진단 & 적정 체중 플래너

키와 몸무게만 입력하면 **현재 건강 상태를 진단**하고, 키에 최적화된 **적정 체중**을 알려주며,
현재 체중에서 적정 체중까지 빼기 위한 **유형별 맞춤 감량 플랜**을 제시하는 웹 앱입니다.

## ✨ 주요 기능

1. **건강 상태 진단**
   - BMI(체질량지수) 계산 및 시각적 게이지 표시
   - 대한비만학회/아시아-태평양 기준 분류 (저체중 · 정상 · 과체중 · 비만 1~3단계)
   - 기초대사량(BMR) 및 하루 유지 칼로리(TDEE) 추정

2. **적정 체중 제시**
   - 키에 최적화된 적정 체중(BMI 22 기준)
   - 건강 체중 범위(BMI 18.5~22.9)
   - 목표까지 필요한 감량/증량량

3. **유형별 감량 플랜** — 각 유형마다 예상 소요 기간을 자동 추천
   - ⚡ **빨리 빼기** — 안전 상한 내 최대 속도
   - 🚶 **적당히 빼기** — 균형 잡힌 속도
   - 🌿 **여유 있게 빼기** — 부담 없는 지속형

   각 플랜은 다음을 구체적으로 제시합니다:
   - 🏃 운동법 (유산소·근력·HIIT·일상 활동 목표 걸음 수 등)
   - 🔥 **소비(소모) 칼로리** — 목표까지 총 소모 칼로리, 칼로리 적자를 식이/운동으로 나눈 분배, 체중 기반 운동별 30분 소모량과 하루 운동 목표를 채우는 데 걸리는 시간
   - 🥗 식이요법 (목표 섭취 칼로리·단백질량·물 섭취량·식사 순서 등)
   - ⏰ 매일 루틴 (아침/낮/저녁/수면/기록/마인드셋)
   - 🗓️ 요일별 주간 루틴 표

## 🧮 계산 근거

- **BMI** = 체중(kg) / 키(m)²
- **적정 체중** = 22 × 키(m)²
- **BMR** = Mifflin-St Jeor 공식
- **감량 계산** = 지방 1kg ≈ 7,700kcal, 안전 감량 속도(주당 체중의 0.4~1%) 기준
- **운동별 소모 칼로리** = MET × 3.5 × 체중(kg) / 200 (분당 kcal)
- 섭취 칼로리는 기초대사량 아래로 내려가지 않도록 보정

> ⚠️ 본 결과는 참고용 추정치이며 의학적 진단·처방이 아닙니다. 건강 상태에 따라 전문의와 상담하세요.

