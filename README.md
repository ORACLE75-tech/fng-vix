# 🌤️ 내일 날씨 준비 (Tomorrow Weather Prep)

출근·여행 **전날 밤**에 내일 날씨(기온·체감·습도·비·구름·바람·자외선)를 미리 확인하고,
**옷차림·우산·챙길 것·유의사항**을 심층적으로 정리해주는 웹앱입니다.

## ✨ 기능

- **도시 검색 + 자동완성** (한국어 지원) / **현재 위치(GPS)** 로 조회
- **내일** 하루 요약: 최고/최저 기온, 체감온도, 강수확률·강수량, 습도, 구름, 바람, 자외선, 일출/일몰
- **심층 추천 엔진**
  - 체감온도(최저~최고) 기반 **단계별 옷차림** 가이드 (한여름 ~ 혹한 8단계)
  - **일교차**가 크면 레이어링 권장 + 감기 주의
  - **비/눈** 정도에 따라 우산(장우산·접이식)·방수신발·미끄럼방지 안내
  - **습도/바람/자외선/폭염/한파**별 맞춤 유의사항
  - **출근(07~09시)·퇴근(18~20시)** 시간대 강수·체감 강조
- **챙길 것 체크리스트** 자동 생성
- **시간대별(3시간 간격) 예보** 카드

## 🚀 실행 방법

별도 빌드·설치가 필요 없습니다. 정적 파일이라 그냥 브라우저로 열면 됩니다.

```bash
# 가장 간단한 방법: 로컬 서버 실행
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

> 현재 위치(GPS) 기능은 보안상 `http://localhost` 또는 `https://` 환경에서만 동작합니다.

## 🛠 기술

- 순수 **HTML / CSS / JavaScript** (프레임워크·빌드 도구 없음)
- 데이터: [Open-Meteo](https://open-meteo.com/) — **무료, API 키 불필요**
  - 예보 API + 지오코딩(도시 검색) API

## 📁 구조

```
index.html   # 화면 구조
styles.css   # 다크 테마 스타일 (모바일 대응)
```
