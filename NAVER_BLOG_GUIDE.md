# 📌 네이버 블로그에 "내일 날씨 준비" 앱 올리기

## 먼저 알아둘 점 (중요)

네이버 블로그 본문은 **보안상 사용자가 만든 JavaScript / 외부 iframe을 실행하지 못합니다.**
즉, 이 앱(자바스크립트로 동작)을 블로그 글 *안에서 직접 화면이 뜨도록* 박아 넣는 것은
네이버 정책상 불가능합니다. (티스토리·워드프레스 등은 가능)

대신 가장 확실하고 표준적인 방법은:

> **앱을 인터넷 주소(URL)로 배포해 두고, 블로그 글에 그 주소를 넣어
> "링크 카드"로 만들기 → 독자가 누르면 앱이 새 창에서 실행됩니다.**

이 저장소는 이미 그렇게 쓸 수 있도록 준비돼 있습니다(무료, HTTPS, 모바일 지원).

---

## 1단계 — GitHub Pages로 무료 배포하기 (1회만)

1. 깃허브에서 이 저장소(`ORACLE75-tech/fng-vix`)로 이동
2. 상단 **Settings** → 왼쪽 메뉴 **Pages**
3. **Build and deployment → Source** 를 **`Deploy from a branch`** 로 선택
4. **Branch** 드롭다운에서 **`claude/weather-prep-app-2slfwb`** 선택, 폴더는 **`/ (root)`** → **Save**
5. 1~2분 기다린 뒤 같은 화면 위쪽에 공개 주소가 나타납니다:

   ```
   https://oracle75-tech.github.io/fng-vix/
   ```

   이 주소를 브라우저에 넣으면 앱이 실제로 동작합니다.

> 💡 나중에 이 브랜치를 `main`에 합치면, Branch를 `main`으로 바꿔도 됩니다.
> 주소(`.../fng-vix/`)는 그대로 유지됩니다.

---

## 2단계 — 네이버 블로그 글에 넣기

### 방법 A. 링크 썸네일 카드 (가장 보기 좋음, 추천)

1. 네이버 블로그 **글쓰기** 진입
2. 본문에 위 주소(`https://oracle75-tech.github.io/fng-vix/`)를 **그대로 붙여넣기**
3. 잠시 기다리면 제목·설명·썸네일 이미지가 들어간 **링크 카드**가 자동 생성됩니다
   (이 저장소에 넣어둔 Open Graph 정보 + `og-image.png` 썸네일이 표시됩니다)
4. 독자가 카드를 누르면 앱이 새 창에서 열립니다 ✅

### 방법 B. 글자에 링크 걸기

1. "👉 내일 날씨 확인하러 가기" 같은 문구를 입력
2. 그 글자를 드래그해서 선택 → 상단 도구의 **링크(🔗)** 클릭
3. 위 주소를 붙여넣고 적용

### 방법 C. 버튼 느낌의 이미지 + 링크

1. 안내 이미지를 하나 올린 뒤
2. 이미지를 클릭 → 링크 아이콘으로 위 주소 연결

## 자주 묻는 점

- **글 안에서 앱이 바로 실행되게 할 수는 없나요?**
  네이버 블로그는 본문 내 사용자 스크립트/iframe을 허용하지 않아 불가능합니다.
  "링크로 연결"이 네이버 블로그에서 가능한 유일한 정상적 방법입니다.

- **썸네일(미리보기 이미지)을 바꾸고 싶어요.**
  저장소의 `og-image.png`(1200×630)를 원하는 이미지로 교체하고 다시 푸시하면 됩니다.
  네이버가 캐시를 가지고 있으면 반영까지 시간이 걸릴 수 있습니다.

- **'현재 위치' 버튼이 안 돼요.**
  위치 기능은 `https://` 또는 `localhost`에서만 동작합니다.
  GitHub Pages는 https라 정상 동작합니다.

- **다른 무료 배포 수단도 되나요?**
  네. Netlify·Vercel·Cloudflare Pages에 올려도 되고, 그 주소를 똑같이 블로그에 넣으면 됩니다.
